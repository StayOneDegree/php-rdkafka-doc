Installing the extension with PECL

Information for installing this PECL extension may be found in the manual chapter titled Installation of PECL extensions. Additional information such as new releases, downloads, source files, maintainer information, and a CHANGELOG, can be located here: » http://pecl.php.net/package/rdkafka

Linux, Unix, and OS X users may run the following command to install the extension: 

$ sudo pecl install rdkafka

Add the following line to your php.ini file: 

extension=rdkafka.so
