RdKafka\ConsumerTopic::consumeQueueStart

(PECL rdkafka >= 0.9.1)

RdKafka\ConsumerTopic::consumeQueueStart — Pipe a partition to a queue

Description

public RdKafka\ConsumerTopic::consumeQueueStart ( integer $partition , integer $offset , RdKafka\Queue $queue ) : void

Same as RdKafka\ConsumerTopic::consumeStart() but re-routes incoming messages to the provided queue queue. The application must use one of the RdKafka\Queue::consume*() functions to receive fetched messages.

Parameters

partition (integer)

Partition ID

offset (integer)

Offset

queue (RdKafka\Queue)

A RdKafka\Queue instance

Return Values

Returns no value.
