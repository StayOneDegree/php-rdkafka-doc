RdKafka\KafkaConsumer::getAssignment

(PECL rdkafka >= 1.0.0, librdkafka >= 0.9)

RdKafka\KafkaConsumer::getAssignment — Returns the current partition assignment

Description

public RdKafka\KafkaConsumer::getAssignment ( void ) : array

Returns the current partition assignment as set by RdKafka\KafkaConsumer::assign() or by rebalancing.

Parameters

This function has no parameters.

Return Values

Returns an array of RdKafka\TopicPartition

Errors/Exceptions

  • Throws RdKafka\Exception on errors.
