RdKafka\KafkaConsumer::queryWatermarkOffsets — Query offsets for a topic / partition

Description

public RdKafka\KafkaConsumer::queryWatermarkOffsets ( string $topic , integer $partition , integer &$low , integer &$high , integer $timeout_ms ) : void

Query broker for low (oldest/beginning) or high (newest/end) offsets for a partition

Parameters

topic (string)

The topic name.

partition (integer)

The partition.

low (integer)

The oldest/beginning offset for the partition

&high (integer)

The newest/end offset for the partition.

&timeout_ms (integer)

Timeout in millisecounds for this operation.

Return Values

Returns no value.
